Automated Method to deploy Django Project onto heroku

You just need to fork and download this project from github.
Open Terminal and Activate your Virtual Environment(Optional)
Now, Run python deploy_app.py
App will open up. Now just select your proect folder and viola!! The configuration is done.
Then, To deploy your app to heroku run following commands in your terminal
  1. heroku login
  2. git init(Optional)
  3. git add .
  4. git commit -m "your message"
  5. git push heroku master

Manual Steps to Deploy Django Project on Heroku

You need to install some dependencies before doing anything
  1. Activate your Virtual Environment(Optional)
  2. Install the following Dependencies
    1. pip install django
    2. pip install django-heroku
    3. pip install heroku
    4. pip install gunicorn
Now, You Will have do some configurations
  1. Open you project directory
  2. Create a file named 'Procfile' without Extension and Open it
  3. Type web: gunicorn projectname.wsgi --log-file -
  4. Then, Open settings.py
    1. At the top type import django_heroku
    2. At the bottom type django_heroku.settings(locals())
Now the configuration has almost been done. just update requirements.txt using pip freeze>requirements.txt
Then, To deploy your app to heroku run following commands in your terminal
  1. heroku login
  2. git init(Optional)
  3. git add .
  4. git commit -m "your message"
  5. git push heroku master
